# Final-Fantasy-XIV-Randomizer

## Description:
Final Fantasy XIV Randomizer a pour but de choisir pour vous les classes/jobs et les roulette ou donjons pour vous. Ceci sans modifier le jeu pour que l'utilisation reste conforme au condition générale d'utilisation de Final Fantasy XIV pour éviter tout bannissement. Ce logiciel se lancera à part du jeu et vous offrira le plus d'options possibles pour modifier votre amusement sur le jeu. Le logiciel est toujours au début de son développement.
